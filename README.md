# AI From Scratch

A neural network framework built from first principles using NumPy.
This is an improved version of my original work, focused on readability, scalability, and following 
overall good software design principles.

## Goals

* Implement neural networks without high-level libraries
* Understand backpropagation and optimization deeply
* Build a modular and scalable ML framework

## Structure

* `layers/`: Core building blocks (Dense, activations)
* `losses/`: Loss functions (MSE, CrossEntropy)
* `optimizers/`: Optimization algorithms (SGD, Adam)
* `models/`: Predefined architectures
* `training/`: Training loops and utilities
